# Building This Site: An AI-Driven Journey

This portfolio isn't just a static resume; it's a testament to modern web development and the power of AI-assisted engineering. Here's a look under the hood.

## The Tech Stack

I chose tools that prioritize speed, type safety, and developer experience:

- **Vite + React**: The foundation. Vite provides lightning-fast builds and Hot Module Replacement (HMR).
- **TypeScript**: Ensuring scalability and catching bugs before they even hit the browser.
- **Tailwind CSS**: A utility-first approach that allowed for the "liquid glass" and high-contrast terminal aesthetics.
- **Marked + Gray-Matter**: Powering the markdown blog system you're reading right now.

## AI-First Development

This entire site was built in collaboration with **Antigravity**, an agentic AI IDE. Instead of just writing code, I worked as an architect, defining high-level goals while the AI handled:

1.  **UI/UX Implementation**: Translating descriptions like "liquid glass" and "smooth rising animations" into precise CSS and Framer Motion logic.
2.  **System Automation**: Creating the Node.js scripts that automatically sync blog metadata and GitHub contributions.
3.  **Refactoring**: Moving from hardcoded sections to a modular, markdown-based content system.

The speed of development was unparalleled—what would usually take days of manual boilerplate was accomplished in a few hours of high-level pair programming.

## Infrastructure & Deployment

The site is hosted on **Vercel**, providing:

- **Seamless CD**: Every push to the main branch triggers an automatic build and deployment.
- **Edge Performance**: Fast global delivery with optimized asset caching.
- **Vercel Analytics**: Real-time insights into performance and user engagement.

## Conclusion

This project shows what's possible when you combine robust engineering foundations with the latest in AI-assisted tooling. It's built for speed, built for the future, and built for the love of the game.
